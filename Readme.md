# FreeRTOS Learning Projects

## Goals

Learn FreeRTOS fundamentals on ESP32 as a stepping stone toward robotics and
embedded systems — concurrent sensor management, motor control loops, and
communication systems all lean heavily on RTOS concepts. Working through
DigiKey's "Introduction to RTOS" video series by Shawn Hymel, one part at a time.

## Tools

- **Board:** ESP32 DevKit (simulated)
- **Framework:** Arduino (via PlatformIO)
- **Simulator:** [Wokwi](https://wokwi.com) — browser-based and VS Code extension
- **Language:** C/C++ (Arduino-style APIs)
